# Library Management System

A Python-based Library Management System that demonstrates the use of data structures and algorithms.

## Features
- Book management (add, remove, search)
- User management
- Book lending and returning
- Search functionality with efficient algorithms

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python src/main.py`

## Data Structures Used
- Hash Tables for quick lookups
- Linked Lists for managing book checkouts
- Binary Search Trees for efficient searching

## Algorithms Implemented
- Quick Sort for sorting books
- Binary Search for fast lookups
- Hashing for user authentication
